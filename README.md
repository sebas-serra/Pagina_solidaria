# 🤝 Acciones Solidarias La Salle

## Descripción
Plataforma web de la comunidad lasaliana para centralizar y visibilizar sus iniciativas de voluntariado y acción social.
Desde una página principal con menú de categorías, el usuario accede a cuatro áreas temáticas —Alimento, Hogar, Social y Educación—
donde puede explorar los distintos planes solidarios disponibles y registrar los retos que ha llevado a cabo.

## Contexto
Este proyecto nació como práctica de desarrollo web con HTML y CSS dentro del Desafío APS (Aprendizaje y Servicio) de La Salle,
del cual resultó ser el proyecto ganador.
El objetivo era crear una web informativa, visualmente cuidada y responsive que sirviera como punto de encuentro
para el voluntariado de los centros lasalianos y facilitara el registro de acciones solidarias mediante un formulario externo de Google Forms.

## Tecnologías usadas
- **HTML5** — Estructura semántica de las páginas principal y de categorías
- **CSS3** — Estilos, animaciones y diseño responsive con Grid y Flexbox
- **JavaScript ** — Renderizado dinámico de planes por categoría y animaciones de transición
- **Git / GitHub** — Control de versiones
